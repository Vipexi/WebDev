Exercise: Modal with Close Button

Add CSS code to make your box look about the same as in image below.

You can vertically center your box by setting margin: auto for your box (.alert).

Note that in order to move the close button (.alert-close) into it's right position you should use the position: absolute CSS property and then top and right properties. In order to make the position: absolute to work, you should set your main container (.alert) to be position: relative.

Best way to also limit your box not be too long is to set max-width or width CSS property for it.

Background color for the alert is #fff3cd, color is #664d03 and border color is #664d03.
